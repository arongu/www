Introduction: This page is an entrance to my fun and sometimes useful frontend projects.¯\_(ツ)_/¯ They are built from scratch, with JS/CSS/HTML/love and with a hint of CSS frustration for browsers interpreting CSS slightly differently. This beautiful menu replicates the Grand Unified Bootloader, you might have seen it on a Linux/BSD system. If not then you are a normal person - You didn't miss anything, it's blue.
I built in a music player as an Easter egg so the letters 'e', 'c' have real functionality, and to prove terminal menus are not necessarily boring. Enjoy!
c: play/pause, e/E: next/prev track, +/-/wheel: volume up/down,
l/L: toggle playlist loop/single track loop, s: toggle shuffle
